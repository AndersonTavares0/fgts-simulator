Opções avançadas

Saldo FGTS

Salário Bruto × 8% × meses trabalhados, com correção monetária estimada (TR + 3% a.a. ou IPCA como piso conforme ADI 5090).

Multa Rescisória

40% para dispensa sem justa causa; 20% para acordo comum (Art. 484-A CLT); 0% para demissão voluntária e justa causa.

13º Proporcional

Salário × (meses trabalhados no ano ÷ 12). Limitado a 12 avos.

Férias Proporcionais + 1/3

Salário × (meses ÷ 12) + um terço constitucional adicional.

Política de Privacidade — LGPD

1. Controlador dos Dados

Este projeto é desenvolvido como atividade acadêmica de Extensão Universitária pelo autor Anderson Tavares, vinculado ao Centro Universitário Internacional UNINTER.

2. Dados Coletados e Finalidade

Este simulador coleta exclusivamente os dados necessários para o cálculo das estimativas de FGTS e rescisão trabalhista:

  • Salário bruto — base para cálculo de depósitos e verbas
  • Datas de início e término — cálculo de meses trabalhados
  • Motivo da rescisão — aplicação da multa correta
  • Tipo de contrato — definição da alíquota FGTS
  • Dados de saúde (doença grave) — dado sensível (LGPD Art. 5, II), tratado com consentimento explícito (Art. 11) apenas para simulação

3. Base Legal

O tratamento de dados baseia-se no consentimento do titular (LGPD Art. 7, I) e no legítimo interesse de fornecer uma ferramenta educacional (Art. 7, IX). Dados sensíveis de saúde são tratados mediante consentimento específico (Art. 11, I).

4. Processamento Local

Todos os cálculos são realizados exclusivamente no navegador do usuário (client-side). Nenhum dado é transmitido, armazenado ou processado em servidores externos. Após o fechamento da página, os dados de simulação são descartados.

5. Armazenamento Mínimo

A única persistência realizada é a preferência de tema (claro/escuro) no localStorage do navegador, sem qualquer relação com dados pessoais ou de simulação.

6. Direitos do Titular (LGPD Art. 18)

Você tem direito a:

  • Acesso — saber quais dados são tratados
  • Correção — retificar dados incompletos ou desatualizados
  • Eliminação — solicitar a exclusão de dados (não aplicável, pois nada é armazenado em servidor)
  • Portabilidade — transferir dados a outro serviço
  • Revogação do consentimento — a qualquer momento, basta fechar a página

7. Segurança

A arquitetura client-side elimina riscos de vazamento em transmissão ou armazenamento em nuvem. Não há backend, banco de dados ou API.

8. Cookies e Tecnologias Similares

Este site utiliza apenas localStorage para persistir a preferência de tema. Não utiliza cookies de rastreamento, analytics ou publicidade.

9. Recursos de Terceiros (CDNs)

Para garantir a melhor experiência visual e performance, este site carrega fontes (Google Fonts) e ícones (Lucide via Unpkg) de provedores externos. Ao acessar o simulador, seu endereço IP pode ser processado por esses provedores exclusivamente para a entrega técnica desses ativos.

10. Contato

Para questões sobre privacidade, entre em contato através do repositório do projeto no GitHub.
